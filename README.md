# Hello World CI/CD Project

## Overview
This project demonstrates:
- A simple Node.js Hello World application
- Docker containerization
- CI/CD pipeline using GitHub Actions
- Automatic push to Docker Hub

## Steps Taken
1. Created a Node.js application
2. Wrote a Dockerfile to containerize it
3. Built and tested locally using Docker Desktop
4. Created a GitHub Actions pipeline
5. Pushed image to Docker Hub

## Run Locally
docker build -t <your-username>/hello-world-cicd .
docker run -p 3000:3000 <your-username>/hello-world-cicd

## Docker Hub Repository
https://hub.docker.com/r/<your-username>/hello-world-cicd
